## Summary

Implement Favorites as a real station-list source backed by persisted `Settings::favorites`.

## Scope

- Populate `ListSource::Favorites` from the persisted `Favorites` collection.
- Applying Favorites replaces `visible` with saved favorite stations and resets/clamps selection.
- Toggling favorite while the active source is Favorites immediately refreshes the visible list.
- Removing the selected favorite clamps to the next valid row, previous row, or empty state.
- Preserve existing favorite identity/deduplication rules from `settings::Favorites`.
- Do not focus on visual empty-state polish in this slice.

## Acceptance Criteria

- Persisted favorites are reachable and playable from the Favorites source.
- Removing a favorite in Favorites source removes it immediately from `visible`.
- Selection remains valid after removing first, middle, last, or only favorite.
- Empty Favorites is represented without falling back to All Stations.
- Reducer/settings tests cover persisted favorites and removal behavior.
