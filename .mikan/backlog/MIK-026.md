## Summary

Add the FFT-based `PeakDots` visualizer mode on top of the shared pane-width sampling helper.

## Scope

- Implement `PeakDots` as a distinct renderer from SpectrumStack.
- Use FFT bands from `VizFrame`, theme spectrum colors, and full pane-width interpolation.
- Render safely in Wide, Medium, and Compact layouts.
- Route visualizer rendering through the selected `VisualizerMode`.
- Avoid fake animation or mode-specific state not derived from `VizFrame`.

## Acceptance Criteria

- Selecting/cycling to PeakDots changes the rendered visualizer.
- PeakDots uses full pane width and real band data.
- SpectrumStack remains default and still works.
- UI tests cover mode selection and PeakDots rendering with synthetic data.
