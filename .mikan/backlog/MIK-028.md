## Summary

Add the three new theme names and persistence/parser support for the Calm Six-pack theme set.

## Scope

- Add `Solarized`, `Midnight`, and `Sakura` to `ThemeName`.
- Update stable lowercase identifiers and strict parser.
- Preserve lenient unknown-theme fallback to Minimal at settings boundary.
- Update cycling order: Minimal -> Neon -> CRT -> Solarized -> Midnight -> Sakura -> Minimal.
- Update settings roundtrip tests for all six themes.
- Update CLI `--theme` parsing and invalid-theme error text.

## Acceptance Criteria

- All six theme names parse and serialize correctly.
- Unknown persisted theme names still fall back to Minimal without dropping other settings.
- `t` cycle order is documented in tests.
- CLI accepts all six stable lowercase names and rejects unsupported names clearly.
- Existing three themes still work.

## Validation

```bash
cargo fmt --check
cargo test theme
cargo test settings
cargo test cli
cargo test app
cargo check
```
