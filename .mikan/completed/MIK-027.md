## Summary

Implement the waveform/RMS-based visualizer modes: `WaveScope`, `MirrorWave`, and `AmbientPulse`.

## Scope

- Implement `WaveScope` using `VizFrame::waveform`.
- Implement `MirrorWave` as a symmetrical waveform display.
- Implement `AmbientPulse` using RMS and/or bands for a low-noise ambient display.
- Ensure all three modes use full pane width via waveform/band interpolation.
- Keep visual intensity calm enough for work-session BGM.
- Update footer/README/docs for the completed five-mode set.

## Acceptance Criteria

- All five visualizer modes are selectable via `v`.
- WaveScope and MirrorWave visibly use waveform data.
- AmbientPulse is real-data-driven and not fake animation.
- Wide, Medium, and Compact layouts render every mode without overlap.
- Tests cover each mode with synthetic `VizFrame` data.
- README/docs describe the modes and `v` control.
